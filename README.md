# Blog-des-Livre
# 📚 Blog de Livres

Un blog interactif permettant de consulter les détails de chaque livre et de partager des commentaires.  
Ce projet a pour objectif de créer un espace convivial pour découvrir de nouvelles lectures et échanger des avis avec les autres utilisateurs.

---

#Lien en Ligne : 
https://imzilnekawfst-design.github.io/Blog-des-Livre/ 

---

## 🧠 Fonctionnalités
- Affichage des détails de chaque livre : titre, auteur, description  
- Ajout de commentaires et partage d’avis  
- Interface simple et intuitive pour faciliter la navigation  

---

## ⚙️ Technologies utilisées
- HTML  
- CSS  
- JavaScript  
- GitHub pour la gestion de version  
- VS Code comme environnement de développement  

---

## 💡 Objectifs pédagogiques / professionnels
- Développer des compétences en **développement web front-end**  
- Mettre en pratique la gestion des interactions utilisateur et du DOM  
- Créer un projet **interactif et partageable** pour un portfolio
